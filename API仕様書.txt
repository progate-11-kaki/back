
1. ユーザー登録
	•	エンドポイント: /register
	•	メソッド: POST
	•	リクエスト:
	•	username (string, 必須): ユーザー名
	•	password (string, 必須): パスワード
	•	password2 (string, 必須): 確認用パスワード
	•	レスポンス:
	•	成功時: リダイレクト /login	
	•	エラー時: ユーザー名の重複やパスワード不一致のエラーメッセージ

2. ログイン		
	•	エンドポイント: /login
	•	メソッド: POST
	•	リクエスト:
	•	username (string, 必須): ユーザー名
	•	password (string, 必須): パスワード
	•	レスポンス:
	•	成功時: 200 OK とホーム画面へのリダイレクト
	•	失敗時: エラーメッセージ表示（ユーザー名またはパスワードが無効です。）

3. プロフィール表示・更新
	•	エンドポイント: /profile
	•	メソッド: GET / POST
	•	認証: 必須
	•	リクエスト:
	•	profile_image (file, 任意): プロフィール画像のアップロード
	•	レスポンス:
	•	成功時: 200 OK プロフィールページを表示
	•	エラー時: ファイルアップロード失敗時にエラーメッセージを表示

4. プロジェクト作成

	•	エンドポイント: /makeproject
	•	メソッド: POST
	•	認証: 必須
	•	リクエスト:
	•	project_name (string, 必須): プロジェクト名
	•	project_description (string, 必須): プロジェクトの説明
	•	tags (string, 必須): カンマ区切りのタグ
	•	commit_message (string, 必須): 最初のコミットメッセージ
	•	commit_image (file, 任意): コミットの画像
	•	レスポンス:
	•	成功時: プロジェクト詳細ページへのリダイレクト
	•	エラー時: 画像アップロード失敗や入力不備によるエラーメッセージ

5. プロジェクト詳細表示

	•	エンドポイント: /project/<int:project_id>
	•	メソッド: GET
	•	認証: 必須
	•	リクエスト: なし
	•	レスポンス:
	•	成功時: プロジェクトの詳細情報および最新コミットを含むページを返す

6. プロジェクトへのユーザー招待

	•	エンドポイント: /project/<int:project_id>/invite
	•	メソッド: POST
	•	認証: 必須
	•	リクエスト:
	•	username (string, 必須): 招待するユーザー名
	•	レスポンス:
	•	成功時: 招待通知が送信された旨のメッセージ
	•	エラー時: ユーザーが見つからない場合にエラーメッセージ
